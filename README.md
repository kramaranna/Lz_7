## Інтерфейс програми

Ось короткий опис того, як я реалізувала цей проєкт:

1. Спроектувала інтерфейс: додала на форму три таблиці DataGridView для візуалізації масиву на різних етапах та три кнопки для керування процесом.

3. Оголосила масив: у коді заголовного файлу створила глобальну матрицю mat[n][n] розміром 3х3 для зберігання цілих чисел.

5. Реалізувала генерацію: перша кнопка очищує таблицю, заповнює масив випадковими числами від 0 до 100 та виводить їх у першу сітку.
  
7. Виконала завдання із заміною: написала алгоритм для другої кнопки, який проходить по всій матриці, знаходить найменше значення та міняє його місцями з останнім елементом у правому нижньому куті, виводячи результат у другу таблицю.

9. Налаштувала сортування: за допомогою циклів в третій кнопці зробила так, щоб у парних рядках (індекс 1) числа ставали за зростанням, а у непарних (індекси 0,2) — за спаданням, після чого оновлена матриця з’являється у третій таблиці.
